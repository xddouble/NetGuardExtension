Privacy Policy

NetGuard Enterprise Security Monitor is designed for managed enterprise and educational environments.

This extension does not collect, store, or transmit personal data to external servers.

All browsing activity evaluation occurs locally on managed devices as part of organization-defined web access policy enforcement.

No user-identifiable data is shared with third parties.

The extension requires access to visited URLs solely for the purpose of evaluating navigation events against enterprise security policies.

This extension is deployed only by authorized system administrators.
